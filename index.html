Sistema de Reserva de Libros

PHP · MySQL · HTML · CSS

Aplicación Web de Reserva de Libros. Este repositorio contiene el código de una aplicación web diseñada para la reserva y gestión de un catálogo de libros. La aplicación soporta tres roles de usuario distintos para un control de acceso granular.

  • Buscador de libros por título o autor.
  • Panel de administración de préstamos.
  • Control de stock y disponibilidad en tiempo real.

Sistema de reservas para peluquería

PHP · MySQL · HTML · CSS

Aplicación web que permite gestionar reservas de citas para una peluquería: selección de tipo de corte, tinte, peinado y fecha. Incluye validación de disponibilidad en la base de datos y registro de reservas.

  • Formulario de reserva con selección de servicios.
  • Comprobación de disponibilidad por fecha en MySQL.
  • Inserción de reservas en tabla reservas.
